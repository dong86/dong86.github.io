Vue 3 - The Complete Guide(incl.Router,Vuex,Composition API)

糖糖爸
2021-05-26
Composition API, JavaScript, Vue 3, Vuex

Learn Vue.js - in its latest version - step by step from scratch. Learn how to build highly reactive web applications using one of the most popular frameworks!

No matter what metric you’re looking at (Google Trends, Github Stars, Tweets …) - VueJS is the shooting star in the world of JavaScript frameworks - it’s just awesome!

This course teaches the latest version of Vue (Vue.js 3) in detail and from scratch. We’ll cover all the basics, but we won’t stop there - this course also delves into advanced concepts such as the Composition API introduced in Vue 3.

This course covers everything!

We’ll start with the very basics of what Vue.js is and how it works before we move on to more advanced and advanced topics, but I’ll be honest: this is too much to fit into one sentence, so this is what you’re in. In this course you will learn:

  • What is VueJS and why use it?
  • The basics (including basic syntax, understanding patterns, and more!)
  • How to display reactive data and listen to events
  • DOM interaction (list rendering, conditionally attaching / detaching elements …)
  • Setting up the development environment and workflow
  • Using components (and which components to start with)
  • A deep look behind the scenes in Vue
  • Binding to form input
  • Sending HTTP Requests to Server APIs
  • Authentication and Vue
  • How to make your app prettier with animations and transitions
  • How to Build a Great Single Page Application (SPA) with Routing
  • How to improve state management with Vuex
  • How to deploy the application
    And much more …

All of these topics are taught through real-world examples and demo projects. This course is full of exercises, challenges, and larger running projects that allow you to practice what you have learned.

Along with many other small projects, we will build a couple of large main projects:

Monster Slayer is a browser-based game 100% powered by Vue.
Learning Resource Manager is a web-based application that allows you to manage (add, remove, list) your favorite learning resources.
Find a Coach Application is a web application that allows you to register / login, register as a coach, list and filter available coaches, and send messages to coaches.
